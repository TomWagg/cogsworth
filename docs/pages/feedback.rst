    .. grid-item-card::

        :card-header-tom:`Providing feedback`

        Did you find a bug in ``cogsworth``? Do you have a feature request? Whatever the case, I'd love to

        .. rst-class:: icon-link

            :fas:`bug` `Report a bug <https://github.com/TomWagg/cogsworth/issues/new?assignees=&labels=bug&projects=&template=bug_report.md&title=>`_

        .. rst-class:: icon-link

            :fas:`book` `Highlight a documentation problem <https://github.com/TomWagg/cogsworth/issues/new?assignees=&labels=documentation&projects=&template=documentation-problem.md&title=>`_

        .. rst-class:: icon-link

            :fas:`lightbulb` `Request a new feature <https://github.com/TomWagg/cogsworth/issues/new?assignees=&labels=enhancement&projects=&template=feature_request.md&title=>`_

    .. grid-item-card::

        :card-header-tom:`Contributing code`

        Do you want to fix a bug you've found? Or perhaps you want to add a new feature?
        Whatever the case I'd love to see your contributions! Please fork the ``cogsworth`` repository and
        create a pull request with your suggested changes when you're ready.

        .. rst-class:: icon-link

            :fas:`code-fork` `Fork the repo <https://github.com/TomWagg/cogsworth/fork>`_

        .. rst-class:: icon-link
