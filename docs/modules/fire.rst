***************************
FIRE simulations (``fire``)
***************************

The ``fire`` module contains the functionality necessary for running simulations that use FIRE simulations
as their initial conditions. This includes the ability to read in FIRE snapshots, find the stellar centre,
estimate the galactic potential and run a full ``cogsworth`` Population based on all that.
