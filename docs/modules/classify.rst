************************************
Binary classification (``classify``)
************************************

``classify`` contains functions for classifying the stars and binaries that are produced in a ``cogsworth`` 
:class:`~cogsworth.pop.Population`. The main functionality is in 
:func:`~cogsworth.classify.determine_final_classes`, but you can also find the full list of classes in 
:func:`~cogsworth.classify.list_classes`.
